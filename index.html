How to push code in github

Follow the steps

  1. Create an account on Github
  2. Create a repository
  3. Install Git SCM on you Computer
  4. Open your project/webpage folder with VS Code
  5. Open Terminal & go to your project directory
  6. Now type the following code on your terminal
    1. git init
    2. git add .
    3. git commit -m "Your Messages"
    4. git branch -M main
    5. git remote add origin https://github.com/username/repository-name.git
    6. git push -u origin main
  7. Your code should be uploaded on your repository
  8. Now you all have to do is create your projects a little bit and type the following code on your terminal
    1. git add .
    2. git commit -m "Your Messages"
    3. git push
